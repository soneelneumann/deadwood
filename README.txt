If you are having troubles running the game, compile all other files before you compile Deadwood.java.

When entering user input, keep in mind that the input is case sensitive. For example, if prompted to "move", typing "Move" will not result in a move and you will be prompted again.
When entering locations or accepting roles, make sure you type the location\role exactly how it appears on the screen.

If you want any information about a room that you are currently in, typing "where" will result in the display of information, including prices of ranks when in Casting Office.

The tutorial at the beginning of the game explains most of the general rules.

Stats shows your rank, money, and credits.
